# Charles Langlan - IT Portfolio

Welcome to my personal IT portfolio website! This site showcases my skills, projects, and experience in Information Technology.

## About

I'm an Information Technology student at UMass Boston with a passion for networking, system administration, and software development. This portfolio highlights my journey and accomplishments in the IT field.

## Features

- **Modern Design**: Clean, responsive design with modern CSS and Font Awesome icons
- **Skills Showcase**: Organized display of technical skills across different IT domains
- **Project Gallery**: Featured projects with descriptions and technology stacks
- **Contact Information**: Easy ways to get in touch via email, LinkedIn, and GitHub
- **Responsive Layout**: Optimized for desktop, tablet, and mobile devices

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Font Awesome Icons
- Responsive Design

## Local Development

To run this site locally:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.

## Deployment

This site is hosted on GitHub Pages. Any changes pushed to the `main` branch will automatically deploy.
